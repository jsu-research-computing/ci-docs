# Training

JSU provides High-Throughput Computing (HTC) training for new users.

## Training modules
- Module 1: Intro to HTC + terminology
- Module 2: Linux / command line basics
- Module 3: HTCondor job scheduling & submit files
- Module 4: Multiple jobs & data transfer
- Module 5: Software environments & reproducibility

Hands-on exercises and examples should live in the `htc-training` and `htcondor-examples` repositories.
