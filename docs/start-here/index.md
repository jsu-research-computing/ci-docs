## New to JSU Research Computing?

Follow this quick path:

1) **Request Access**  
Start the onboarding process and learn what you need to get set up.  
➡ {{ site.baseurl }}/get-access/

2) **Complete Training**  
Work through the training modules before running jobs on shared resources.  
➡ {{ site.baseurl }}/training/

3) **Run Your First Job (Example)**  
Use copy/paste starter workflows (hello-world, file transfer, DAGMan).  
➡ **Open the examples repo:** https://github.com/jsu-research-computing/htcondor-examples

4) **Review Policies**  
Make sure you understand acceptable use and security expectations.  
➡ https://github.com/jsu-research-computing/ci-policy

5) **Get Help if Needed**  
Troubleshooting and support resources.  
➡ {{ site.baseurl }}/support/
