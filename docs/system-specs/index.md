## JSU Research Computing System Specs

This page provides a simple overview of the current JSU research computing hardware used for access and execution.

---

## Access Point

The access point is the system users connect to for submitting and managing jobs.

This node supports:
- user access
- job submission
- workflow management
- onboarding and training use cases

---

## Execution Point

The execution point is the system where research workloads actually run.

This node supports:
- HTCondor job execution
- shared compute workloads
- high-throughput computing activity
