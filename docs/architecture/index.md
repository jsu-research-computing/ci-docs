## Computing Infrastructure

Researchers run workloads through distributed computing systems such as:

• HTCondor clusters  
• Open Science Grid (OSG)  
• OSPool resources  

These platforms allow high-throughput computing workflows to run across shared infrastructure.

---

## Why this structure works

Separating the **portal interface** from the **content repositories** makes the system:

• easier to maintain  
• version controlled  
• scalable for new resources  
• easier for researchers to navigate
